Magnezone

Magnezone, or Sheriff Magnezone as he is officially called, is a Discord bot made for the Pokemon Crossroads Discord server.

This project was made in combination with Discord.js and Discord.js-commando. The documentation website was made with VuePress and is hosted on Netlify.

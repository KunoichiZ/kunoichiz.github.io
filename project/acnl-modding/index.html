ACNL Modding Info

This project was an idea that I had for an Animal Crossing New Leaf (ACNL) Modding/Hacking Discord server.
I was inspired to make the site because the information in the Discord server was all over the place and really needed a more general place.

This project was made in combination with Nuxt.js, Vue.js, Bulma, Buefy, FontAwesome, and node-sass. It is hosted on GitHub Pages.

A link to the project can be found here: ACNL Modding Info
